Independent - Click 2.0 (On pause)

MMORPGs always state that using software to click and do things for you is illegal. I have always wondered, is it possible to make a hardware based work around this law ? I set out to do this with click 2.0, a rework of my previous auto clicker robot (click 1.0) Note: project on pause until I am done with undergrad university and I also need money for a power supply

Summary

Objectives:

Click 2.0 is a machine that should have the ability to:

  • Identify objects and numbers on a computer screen via webcamera
  • Interact with those identified objects to complete some action
  • Be run independent of the computer that is displaying information

Challenges:

  • Prototyping the mechanical components should be easy but the later objectives present challenges.
  • I have limited coding experience and would need to improve to accomplish image detection using openCV
  • I intend to use a raspberry pi 4 to act as the second independent computer and need to figure out how to use the I/O ports to control the motor

Results:

  • A refined version of the CAD model that can move the mouse across the screen.
  • A first prototype that has proven to move the mouse with precision
  • Code in progress that will first identify objects and do the clicking via code, and then transition to hardware based clicking
